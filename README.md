# 金融工程个人主页与量化分析可视化系统

欢迎来到我的金融工程个人项目仓库。本项目构建了一个集成了**个人履历展示**、**实时股价蒙特卡洛模拟**、**A股量化择时回测**以及**马科维茨有效边界优化**的交互式前端量化分析系统。

## 🚀 系统核心模块

本仓库由以下四大核心模块组成，所有量化模型均实现前端轻量化交互与高性能可视化：

1. **个人主页 & 实盘量化引擎 (`index.html`)**
   * **动态主页**：支持中英双语切换的个人全栈简历与项目门户。
   * **蒙特卡洛预测引擎**：支持接入 Live Yahoo API（具备 GitHub Actions 缓存容灾机制），通过几何布朗运动（GBM）生成逐日归一化的概率密度热力图，实时模拟并预测美股与A股的价格路径。

2. **A股量化回测模型 (`backtest.html`)**
   * **多维指标融合**：基于宏观流动性指标与市场情绪指标构建的择时交易策略。
   * **可视化回测**：支持自定义回测周期与参数配置，动态输出策略资产净值曲线（NAV）、最大回撤、夏普比率（Sharpe Ratio）等核心回测指标。

3. **现代资产配置优化模型 (`efficient_frontier.html`)**
   * **马克维茨均值-方差模型 (MPT)**：输入多元资产的历史收益率与协方差矩阵，在线求解切点组合（Tangency Portfolio）与最小方差组合（GMV）。
   * **有效边界可视化**：利用 ECharts 绘制高维散点图，直观展现风险-收益前沿边界、资本市场线（CML）及单资产分布。

4. **个人履历 (`王盛烨简历.pdf`)**
   * 包含上海纽约大学（商业与金融·商业分析方向）的学术背景、量化实习经历及核心技能栈（Python / 计量经济学 / 机器学期 / 金融建模）。

---

## 🛠️ 技术栈

* **前端开发**：HTML5, CSS3 (Tailwind 风格响应式布局), Vanilla JavaScript (ES6+)
* **数据可视化**：ECharts.js (高性能图表库), Font Awesome 6.0
* **量化算法**：几何布朗运动 (GBM) 随机微分方程求解、马科维茨均值-方差优化
* **数据源接口**：Yahoo Finance API & GitHub Pages 每日静态数据缓存
